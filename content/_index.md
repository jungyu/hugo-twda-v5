# 歡迎來到 Hugo + TailwindCSS + DaisyUI 5

這是一個現代化的靜態網站，展示了 Hugo、TailwindCSS 和 DaisyUI 的完美整合。

## 技術特色

- **Hugo v0.147.9**: 快速的靜態網站生成器
- **TailwindCSS v4.1.11**: 實用優先的 CSS 框架
- **DaisyUI v5.0.43**: 美觀的組件庫
- **Alpine.js v3.14.9**: 輕量級 JavaScript 框架

## 功能亮點

- 🎨 現代化設計系統
- 🌙 深色/淺色主題切換
- 📱 完全響應式設計
- ⚡ 極快的載入速度
- 🔍 SEO 優化
