# 開始使用 Hugo + TailwindCSS + DaisyUI

本文將帶您快速了解如何使用 Hugo、TailwindCSS 和 DaisyUI 建立現代化的靜態網站。

## 為什麼選擇這個組合？

### Hugo
- 極快的建構速度
- 豐富的模板系統
- 強大的內容管理

### TailwindCSS
- 實用優先的設計哲學
- 高度可客製化
- 優秀的開發體驗

### DaisyUI
- 美觀的預設組件
- 豐富的主題選擇
- 與 TailwindCSS 完美整合
