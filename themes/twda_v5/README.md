# TWDA v5 - Hugo Theme

現代化 Hugo 主題，整合 TailwindCSS v4 + DaisyUI v5 + Alpine.js v3

## 技術棧

- **Hugo**: v0.147.9+
- **TailwindCSS**: v4.1.11
- **DaisyUI**: v5.0.43
- **Alpine.js**: v3.14.9

## 特色功能

- 🎨 現代化設計系統
- 🌙 深色/淺色主題切換
- 📱 完全響應式設計
- ⚡ 極快的載入速度
- 🔍 SEO 優化
- ♿ 無障礙支援
